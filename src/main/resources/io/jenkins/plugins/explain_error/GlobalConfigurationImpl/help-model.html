The AI model to use for error explanation.

Recommended models by provider:

  • OpenAI:
    • gpt-4, gpt-4-turbo, gpt-3.5-turbo, etc.
    • More advanced models available https://platform.openai.com/docs/models/
  • Google Gemini:
    • gemini-2.0-flash, gemini-2.0-flash-lite, gemini-2.5-flash, etc.
    • More advanced models available https://ai.google.dev/gemini-api/docs/models
  • Ollama:
    • gemma3:1b, gpt-oss, deepseek-r1, etc. any model available in your Ollama instance
    • More advanced models available https://ollama.com/search

The model you choose affects both the quality of explanations and the API cost. Generally, more advanced models provide better explanations but cost more per request.
