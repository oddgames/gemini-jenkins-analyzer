Regular expressions to filter error logs (one per line). Only log lines matching these patterns will be sent to the AI for analysis.

Priority:

  1. If specified here, these patterns will be used
  2. Otherwise, patterns from global configuration will be used
  3. Otherwise, the single "Log Pattern" field will be used
  4. If none are specified, all logs will be analyzed

Example Unity patterns: 

(?i)\bError\s*:\s*
(?i)\bException\b
(?i)^\s*\*\*\s*BUILD FAILED\s*\*\*
(?i)error:\s*[^\s]

Patterns are case-insensitive by default when using (?i) flag.
